# TestFlightから元データだけを任意移行する

TestFlightから一般公開環境への移行は本人が希望した場合だけ行い、契約、金額、更新日、容量、利用量集計、任意メモを移す。見直し結果、確認優先度、単価、節約額は、古い計算方式や料金情報を引き継がないよう一般公開環境で再計算する。

## Consequences

- セッション、認証・端末同期トークン、問い合わせ、診断情報、アンケート回答、監査ログは移行しない。
- 自動移行と既存アカウントへの自動統合を行わない。
- 移行失敗時はTestFlight側の元データを保持し、再試行またはCSV・JSON出力を選べるようにする。
- 移行完了後も、一般公開環境で見直し再計算が終わるまで古い結果を表示しない。
