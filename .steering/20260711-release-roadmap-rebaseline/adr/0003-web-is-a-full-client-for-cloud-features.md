# Webをクラウド機能の正式クライアントとする

Webは任意の補助画面ではなく、iPhone固有機能を除く主要なクラウド機能を利用できる正式クライアントとする。ただしiPhoneだけでも主要操作を完結できるようにし、TestFlight期間中は招待済みiPhoneで初回有効化したアカウントだけWeb利用を許可する。

## Consequences

- 契約、支出、見直し、更新、容量管理、出力、退会、端末・セッション管理、問い合わせをWebでも提供する。
- Screen Time権限、計測対象アプリ選択、主計測iPhoneの端末内処理、Face IDアプリロックはiPhone専用とする。
- iPhoneとWebの共通機能は同じクラウドデータと認可境界を使い、機能差を対応表で管理する。
