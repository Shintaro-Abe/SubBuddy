# Renderシンガポールでクラウドデータを保存する

根拠のない国内保存制約を撤回し、TestFlightと一般公開のAPI、PostgreSQL、バックアップをRenderのシンガポールリージョンで運用する。SubBuddyは国外保存を隠さず、保存国、データ種類、安全管理、Renderと再委託先の取扱条件を確認して利用者へ説明する。

## Consequences

- TLS、保存時暗号化、通常の本番DB直接アクセス禁止、時間制限・監査付き緊急アクセス、ログ最小化を必須とする。
- 契約、金額、更新日、利用量、見直し結果を外部のログ・監視サービスへ送らない。
- 一般公開前にRenderの契約、データアクセス、再委託、削除、バックアップ、シンガポールの制度と日本法上の取扱いを確認し、必要に応じて専門家確認を行う。
- 保存国または事業者を変更する場合は、プライバシー説明とApp Store Connect申告への影響を再確認する。
