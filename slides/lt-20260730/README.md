## 発表する

`deck.html`をブラウザで開き、全画面表示にする。

| 操作 | キー |
|---|---|
| 次へ | `→`、`PageDown`、Space |
| 前へ | `←`、`PageUp` |
| 最初／最後 | `Home`／`End` |
| 発表者ノート表示 | `N` |

発表者ノートを表示するとスライドが縮むため、投影中は非表示にする。PDFを使う場合は、PDFビューアーの全画面表示でもよい。

## スライドを再生成する

`deck.html`を変更した後、次を実行する。

```bash
node slides/lt-20260730/scripts/render-deck.mjs
```
