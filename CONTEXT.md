# SubBuddy

SubBuddyが、利用者のサブスクリプション支出と見直しを支援するために使う共通言語を定義する。

## クライアント体験

**iPhone向けWeb**:
iPhoneのブラウザで利用するSubBuddy Web。PC版と同じ機能・データを扱いながら、主要移動は「ホーム・契約・見直し」の下部ナビゲーションを使う。ブラウザや端末モデルごとの別製品には分けない。
_避ける表現_: スマホ版アプリ、簡易版Web

**PC向けWeb**:
幅の広いブラウザで利用するSubBuddy Web。主要移動には左サイドバーを使う。
_避ける表現_: デスクトップアプリ

**主要画面**:
日常的に行き来する「ホーム・契約・見直し」の3画面。iPhone向けWebでは下部ナビゲーションから直接開く。
_避ける表現_: メインタブ以外の全画面

**補助画面**:
主要画面の判断や管理を補う「支出の内訳・更新間近・使い方・設定」。iPhone向けWebでは各画面の直接導線に加え、右上の「その他」から開く。
_避ける表現_: 隠し画面、低優先画面
